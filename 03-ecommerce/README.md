# Ecommerce API

1. Clonar proyecto
2. Instalar dependencias

```
pnpm i
```

3. Clonar **.env.template** y renombrar a **.env**
4. Cambiar las variables de entorno
5. Levantar la base de datos

```
docker-compose up -d
```

6. Iniciar modo desarrollo

```
pnpm dev
```

7. Ejecutar Seed

```
http://localhost:3000/api/seed
```
